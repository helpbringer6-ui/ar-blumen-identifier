# AR Blumen Identifier

## Projektbeschreibung
Diese Anwendung ermöglicht es Benutzern, Blumen in ihrer Umgebung durch die Kamera ihres Mobilgeräts zu scannen. Die App zeigt dann Informationen über die identifizierte Blume in Augmented Reality an, einschließlich ihres Namens, ihrer Pflegeanleitung und interessanter Fakten.

## Funktionen
- Scannen von Blumen über die Kamera
- Anzeigen von Kunstinformationen in AR
- Informationen zur Pflege und Herkunft der Blumen

## Technologien
- ARKit (für iOS) oder ARCore (für Android)
- React Native oder Flutter für plattformübergreifende Entwicklung

## Installation
1. Klonen Sie dieses Repository
2. Führen Sie die App auf Ihrem Mobilgerät aus
